# readme

__qml-rotate__ demos how to use qml transform and rotation to rotate an image
both on x-axis and z-axis. Only angle at z-axis changes with timer. Angle at
x-axis is fixed.

## note

may use
```
$ mkdir -p build && cd build && qmake ..
$ make
$ ./qml-rotate
```
or
```
$ qmlscene main.qml
```
to see the result
